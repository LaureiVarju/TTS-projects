Fibonacci Javascript Exercise A

The goal of this code is to pull a specific value from an array that generates the Fibonacci sequence.

Once you input a value in the prompt, the console will display the Fibonacchi number that occurs in the sequence equivalent to the input value.

For example: the first five numbers that occur in the Fibonacci sequence are: 0, 1, 1, 2 and 3.

If you input '5' into the prompt, the value returned in the console will be 3, because 3 is the fifth number in the Fibonacci sequence.
